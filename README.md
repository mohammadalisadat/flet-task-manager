# Flet Task Manager

Flet Task Manager is a personal productivity application developed with Python and Flet.  
It helps users manage tasks, track daily habits, define goals and sub-goals, monitor progress, and visualize personal activity data.

## Features
- Add, edit, and manage personal tasks
- Set task priority and scheduled time
- Track daily habits and mark them as completed
- Define goals and sub-goals
- Monitor goal progress with percentage tracking
- View activity statistics using charts
- Store personal body information and health-related metrics

## Technologies Used
- Python
- Flet
- Charts and data visualization
- Desktop application design

## Use Cases
- Personal task management
- Habit tracking
- Productivity monitoring
- Goal planning and progress analysis
